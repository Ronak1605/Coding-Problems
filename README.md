# Coding Problems
This repository is for storing and showing the Coding problems I have solved in my free-time. Each file represents a 
problem solved. Each file will contain the problem description, solution code, my thought process while solving the problem, and test
cases (if applicable). 

As this is not a project or codebase repository, the best branching, commit, and pull request practices are not going to be used. If you want to see how I use common workflows, please see my other repositories.

The problems will be seperated by language.
